📖 About This Project

How It Works?

When you upload an HTML or CSS file, our system:

  • Parses the HTML/CSS file
  • Analyzes CSS styles (Inline + Stylesheets)
  • Detects all types of color codes (HEX, RGB, RGBA, HSL)
  • Finds named colors (e.g., red, blue, green)
  • Shows preview of each color with copy functionality

Technologies Used

Django 5.0 BeautifulSoup4 Tailwind CSS Python 3.11+ JavaScript ES6

Key Features

  • HTML & CSS file support
  • HEX, RGB, RGBA, HSL color detection
  • Named colors extraction
  • Instant color preview
  • One-click copy to clipboard
  • Responsive design for all devices

Future Features

  • Multiple file upload at once
  • Color palette generation
  • Color contrast checker
  • User accounts & history
  • Export color palette as JSON/CSV
  • Color scheme suggestions

💡 Pro Tip: Click on any color code to automatically copy it to your clipboard!
