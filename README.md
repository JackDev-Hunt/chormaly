# Color Extract Project 🎨

A Django-based web application that extracts colors from images and shows HEX/RGB values.

## Features
- Image upload
- Color extraction
- HEX & RGB display
- Responsive UI

## Tech Stack
- Django
- HTML / CSS / JavaScript
- Tailwind (if used)

## How to Run
pip install -r requirements.txt
python manage.py runserver
